# ぎょぎょっと出すアプリ。







======================
django-simple-calendar
======================

月間カレンダー、週間カレンダーなどの機能を提供します。

`解説ブログ <https://narito.ninja/detail/11/>`_

確認した環境
----------

:Python: 3.6以上
:Django: 2.0以上


使い方
-----
1. インストールする。::

    git clone https://github.com/naritotakizawa/django-simple-calendar
    pip install django

2. 動かす。::

    python manage.py migrate
    python manage.py runserver
